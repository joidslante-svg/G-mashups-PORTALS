# G-mashups - PORTALS

G-mashups - PORTALS is a mossy green Spicetify theme with light pink text.

Install with Spicetify Marketplace after the repo is indexed.

Manual install copies the PORTALS folder to the Spicetify Themes folder, then runs spicetify config current_theme PORTALS color_scheme Base and spicetify apply.
